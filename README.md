# 农业植保专家顾问团 Skill

> 融合4位顶级专家的思维框架，为农业植保公司产品经理提供智能决策支持

---

## 🎯 功能亮点

### 1. **多角色融合** - 4位专家智能切换
- 🌾 **康振生院士**（植物病理学）— 根源分析 + 系统思维
- 🚁 **何雄奎教授**（植保装备）— 枪-子弹-打法 + 飘失一票否决
- 🧪 **宋宝安院士**（绿色农药）— 问题驱动 + 产业化导向
- 🐛 **吴孔明院士**（害虫防治）— 生态系统视角 + 预防性原则

### 2. **智能路由** - 自动匹配最合适的专家
用户输入问题 → 自动识别问题类型 → 匹配最相关的专家回答

**示例：**
- "如何设计病害监测系统？" → 🌾 康振生
- "植保无人机飘失怎么解决？" → 🚁 何雄奎
- "绿色农药创制方向是什么？" → 🧪 宋宝安
- "转基因棉花的安全性如何评价？" → 🐛 吴孔明

### 3. **委员会讨论模式** - 复杂问题多专家协同
当问题涉及多个领域时，自动启动**委员会讨论**：
1. 问题拆解 → 识别涉及的专家领域
2. 并行调用 → 每位专家从自己视角分析
3. 观点整合 → 综合各位专家的结论
4. 输出格式化结果（标注每位专家的贡献）

## 🚀 安装方法

### 方式1：从SkillHub安装（推荐）
```bash
skillhub install agri-plant-protection-advisory
```

### 方式2：本地安装（zip包）
1. 下载 `agri-plant-protection-advisory.zip`
2. 运行安装命令：
```bash
skillhub install ./agri-plant-protection-advisory.zip
```

### 方式3：手动安装
1. 下载Skill包并解压
2. 将 `agri-plant-protection-advisory/` 目录复制到你的OpenClaw workspace：
   ```
   ~/.qclaw/workspace-<your-agent-id>/skills/
   ```
3. 重启OpenClaw Gateway：
   ```bash
   openclaw gateway restart
   ```

## 📋 专家决策启发式（部分示例）

### 康振生院士
1. **根源分析优先** - 先问"为什么会这样？"
2. **系统解决方案** - 构建综合技术体系
3. **证据驱动决策** - 基于实地数据做判断

### 何雄奎教授
1. **枪-子弹-打法必须匹配** - 装备+农药+技术协同
2. **飘失一票否决** - 飘失控制不过关，再炫酷也没用
3. **超前10-20年** - 研究要超前，但"用不用得起"是现实问题

### 宋宝安院士
1. **问题驱动** - 研究课题来源于生产，成果回到生产中去
2. **多学科交叉** - 主动推进生物技术、信息技术跨界协同
3. **产业化导向** - 强调研究成果的实际应用效果

### 吴孔明院士
1. **生态系统视角** - 害虫不是孤立存在的
2. **适应国情** - 不能照搬西方理论，必须适应中国农业结构
3. **预防性原则** - 充分证据积累→分阶段批准

## ⚠️ 诚实边界

本Skill是基于**公开信息**提炼的认知框架，**不是专家本人**，有以下局限：

1. **信息截止**：2026年5月（之后发生的事件/观点不包含）
2. **不适用场景**：纯软件开发、互联网产品（专家经验在农业植保）
3. **可能盲区**：最新AI技术、互联网商业模式
4. **表达DNA**：部分专家的中文语录因网络访问限制未能获取，表达风格为推断
