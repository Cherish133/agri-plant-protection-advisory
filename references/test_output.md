───────────────────────────────────────────────────────────
🌾 康振生院士（植物病理学视角）
───────────────────────────────────────────────────────────
【核心观点】
水稻条纹叶枯病是由灰飞虱传播的病毒病，与普通真菌病害不同，其监测逻辑需要从"病原孢子监测"转向"传毒介体监测+病毒早期检测"。

关键点：
1. **源头治理**：灰飞虱是传毒介体，监测重点是飞虱种群动态而非病斑
2. **时间维度提前**：条纹叶枯病一旦显症已无法治疗，必须在传毒前干预
3. **证据驱动**：需要长期监测数据建立流行预测模型

【决策建议】
1. 监测方案增加"灰飞虱种群监测"模块（灯诱/黄板+图像识别）
2. 建立灰飞虱带毒率检测机制（ELISA/PCR快速检测）
3. 结合气象数据建立流行预警模型（温湿度→飞虱繁殖→发病风险）
4. "控前保后"：重点在水稻分蘖期前控制飞虱

【案例参考】
小麦条锈病"控西保东"策略——从源头控制菌源传播

───────────────────────────────────────────────────────────
🚁 何雄奎教授（植保装备视角）
───────────────────────────────────────────────────────────
【核心观点】
产品设计要遵循"枪-子弹-打法"原则：

- **枪**：无人机/喷雾装备（打）
- **子弹**：药剂+精准剂量（药）
- **打法**：施药参数+时机选择（策略）

条纹叶枯病的产品核心不是"治病"，而是"治虫防病"。

【决策建议】
1. **装备选型**：大田用无人机精准施药，丘陵用地面喷雾机
2. **施药时机**：灰飞虱迁飞峰期提前1-2天用药
3. **飘失控制**：选择中午高温时段施药（飞虱活动高峰期+减少飘失）
4. **用不起得起**：药剂成本控制在每亩15-20元
5. **标准化**：制定无人机防治灰飞虱作业标准

【案例参考】
植保无人机防治水稻病虫害作业规范

───────────────────────────────────────────────────────────
🧪 宋宝安院士（农药创制视角）
───────────────────────────────────────────────────────────
【核心观点】
条纹叶枯病本质是病毒病，目前没有直接杀病毒药剂，核心策略是"治虫断毒"。

绿色农药选择原则：
1. **有效性**：防治灰飞虱（吡虫啉、噻虫嗪、氟啶虫胺腈）
2. **安全性**：低毒、低残留、对天敌友好
3. **抗性替代**：轮换用药，避免抗性过快发展
4. **环境友好**：优先选择生物制剂+化学农药复配

【决策建议】
1. **核心药剂**：噻虫嗪（持效长）+苦参碱（生物制剂）复配
2. **抗性管理**：一季最多用2次，交替使用不同作用机理药剂
3. **16字箴言**：正确认识（病毒病难治）、严格管理（农药使用）、规范使用（剂量时机）、科学发展（持续优化）
4. **营养协同**：加入叶面肥（如氨基酸水溶肥），增强植株抗病力

【案例参考】
茶叶绿色防控技术体系——生物制剂+化学农药协同

───────────────────────────────────────────────────────────
🐛 吴孔明院士（害虫治理视角）
───────────────────────────────────────────────────────────
【核心观点】
从生态系统视角看，灰飞虱-条纹叶枯病不是孤立问题：

1. **生态系统**：灰飞虱种群受天敌（蜘蛛、寄生蜂）制约
2. **时间维度**：需要长期监测数据（不是临时监测）
3. **区域联防**：灰飞虱可迁飞，需要"全国一盘棋"
4. **适应中国结构**：小农户分散种植，不能照搬规模化农场的方案

【决策建议】
1. **生态调控**：保护和利用天敌（减少化学农药对天敌的杀伤）
2. **区域联防**：建立县级/省级灰飞虱监测预警网络
3. **长期规划**：Bt水稻（抗虫）作为长期解决方案的可行性评估
4. **监管路径**：如果集成转基因抗虫技术，需提前规划监管路径

【案例参考】
Bt棉种植区害虫综合治理——长期跟踪、系统思维

📌 综合建议
═══════════════════════════════════════════════════════════

【共识点】
1. **核心是治虫防病**：监测对象是灰飞虱而非病斑
2. **时机是关键**：分蘖期前的预防窗口期至关重要
3. **综合防控**：药剂+装备+生态调控协同
4. **绿色优先**：生物制剂与化学农药复配，减少化学农药用量

【分歧点】
| 专家 | 核心分歧 |
|------|----------|
| 康振生 vs 吴孔明 | 监测重心：病原检测 vs 种群生态监测 |
| 何雄奎 vs 宋宝安 | 药剂策略：成本优先 vs 抗性管理优先 |

【优先级排序】
1. ⭐ **建立灰飞虱监测网络**（康振生+吴孔明共识）——基础数据是预警的前提
2. ⭐ **制定精准施药方案**（何雄奎主导）——执行层面的核心
3. ⭐ **绿色农药配方**（宋宝安主导）——药剂选择决定效果和安全性

【下一步行动】
1. **第1个月**：搭建灰飞虱监测模块（灯诱/黄板+图像识别）
2. **第2个月**：集成气象数据，建立流行预测模型
3. **第3个月**：联合何雄奎团队测试无人机施药参数
4. **第4个月**：联合宋宝安团队确定药剂配方
5. **第5-6个月**：小规模田间验证，优化产品
